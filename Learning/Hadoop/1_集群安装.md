# Hadoop安装

1. 关闭防火墙

   ```
   service iptables stop \\服务关闭
   chkconfig iptables off \\关闭开机自启
   ```

2. 创建一个一般用户

   ```
   useradd evan
   passwd yousr passwd
   ```

   

3. 再/opt目录下创建software module文件夹, 并更改所有权

   ```
   mkdir /opt/softawre /opt/module
   chown evan/passwd /opt/software /opt/module
   ```

   

4. 这个用户加到sudoers

   ```
   vim /etc/sudoers
   root ALL=(ALL) ALL
   evan ALL=(ALL) NOPASSWORD:ALL
   
   su evan
   sudo ls
   exit; 
   ```
