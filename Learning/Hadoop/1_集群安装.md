# Hadoop安装

### 1.linux配置

1. 关闭防火墙

   ```
   service iptables stop \\服务关闭
   chkconfig iptables off \\关闭开机自启
   ```

2. 创建一个一般用户

   ```
   useradd evan
   passwd yousr passwd
   ```

   

3. 再/opt目录下创建software module文件夹, 并更改所有权

   ```
   mkdir /opt/softawre /opt/module
   chown evan/passwd /opt/software /opt/module
   ```

   

4. 这个用户加到sudoers

   ```
   vim /etc/sudoers
   root ALL=(ALL) ALL
   evan ALL=(ALL) NOPASSWORD:ALL
   
   su evan
   sudo ls
   exit; 
   ```

5. 改Hosts

   ```shell
   # host就是一张名单，类似座位表
   # 使用ipconfig命令查看自己的ip
   # 添加所有的IP,每台机器都需要修改
   vim /etc/hosts
   192.168.1.100···hadoop100
   ```

   （每clone一个都在这里修改一次，修改IPSDDR，HOSTNAME）

6. 该静态IP

   ```shell
   vim /etc/sysconfig/network-scripts/ifcfg-eth0
   # 多的信息全部删除
   DEIVCE=eth0
   TYPE=Ethernet
   NOBOOT=yes
   BOOTPROTO=static
   IPSDDR=192.168.1.100 # 和hosts的对应
   PREFIX=24
   GATEWAY=192.168.1.2 # 网关设置
   DNS1=192.168.1.2
   NAME=eth0
   ```

7. 修改主机名

   ```
   #要和hosts的主机名对应
   vim /etc/sysconfig/network
   HOSTNAME=hadoop100
   ```

8. 修改`attr`

   ```
   sudo vim /etc/udev/rules.d/70-persistent-net.rules
   ```
